Main feature

The [content] shortcode displays the following: posts, pages, custom post types, fields, images, attachments, menus, or widget areas. The [loop]Â shortcode performs query loops. It can show, for example, available products in a category, or excerpts from the 5 most recent posts. See an overview of your site's content structure at: Dashboard -> Content

Included

  • Simple gallery field for any post type
  • Relative URLs for links and images
  • Include HTML/CSS/PHP/JavaScript files
  • User name, ID, login/logout links with redirect
  • Display content for admin, specific user, login status
  • Comments list, input form or comment count
  • Display content according to Mobile Detect

With other libraries

  • Advanced Custom Fields - image, gallery, repeater, flexible content
  • Bootstrap carousel, navbar menu, pills and stacked
  • Live Edit - front-end editing of content and fields
