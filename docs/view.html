View

Here is an example of how to display your content. Let's imagine a bicycle shop. Create a custom post type called bicycle, and add custom fields such as model, price, and description. Add all bicycles in the store as new entries, with featured image and other info in the fields. Create a new page to display the bicycles. A basic template could be: [loop type="bicycle"]     [field image]     Model: [field model]     Price: [field price]     Description: [field description] [/loop] Another page can display a list of bicycles in the freestyle category: Freestyle bikes available: <ul>     [loop type="bicycle" category="freestyle"]         <li>             [field title-link]             [field model] - [field price]         </li>     [/loop] </ul>

Edit

You can create and edit these templates inside the post with the Text editor. They can also be loaded from a number of places.
  • Custom post type: [content type="template" name="home-page"]
  • Custom field: [field code_block]
  • Sidebar: Enable shortcodes inside Text widget
  • File: [load dir="views" file="recent-posts.html"]
There are a couple points to keep in mind:
  • The content of the post editor is automatically formatted, which can create unwanted tags inside multi-line shortcodes. Try wrapping the section with the [raw] shortcode to prevent this.
  • Don't use the Visual mode when editing HTML, as the code can sometimes be wrongly interpreted. You can disable the visual editor per page or post type with the plugin Raw HTML, or move the code outside the post content with one of the methods above.
