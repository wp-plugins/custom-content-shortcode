For/each loop

This is a feature to create a loop for each category, tag, or taxonomy term.

Display a loop for each category

[for each="category"]     [each name]     [loop type="post"]         [content field="title"]     [/loop] [/for] It gets all terms of a given taxonomy, in this case category, and loops through each term.

For each category, display 3 most recent post titles with links

[for each="category"]     Category: [each name]     [loop type="post" count="3"]         <a href="[content field='url']">             [content field="title"]         </a>     [/loop] [/for]

Parameters

Available parameters for the [for] shortcode are:
  • each - category, tag, or custom taxonomy slug
  • orderby - order by name (default), count (post count), id, or slug
  • order - ASC (ascending - default), or DESC (descending)
  • count - limit number of terms: count="3"
Available parameters for the [each] shortcode are:
  • name - name of category, tag, or taxonomy term
  • slug - slug of category, tag, or taxonomy term
  • id - ID of category, tag, or taxonomy term

Category/tag/taxonomy of each post

It's also possible to use [for] and [each] inside a loop.

Display a link for each category assigned to a post

[loop type="post"]     [content field="title"]     [for each="category"]         <a href="[url]/category/[each slug]">[each name]</a>     [/for] [/loop]
