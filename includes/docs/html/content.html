Content

Use [content] to display post content or a field.

Display post content by name

[content name="hello-world"]

Display the featured image of a page

[content type="page" name="about-me" field="image"]

Display a field from a custom post type

[content type="apartment" name="lux-suite-22" field="rent-per-day"]

Basic parameters

Post

  • type - post type: post, page, or custom post type - default is any
  • name, title, or id -  get post by name/slug, title, or ID - default is current post

Fields

  • field - name of field to display - if empty, the default is the post content
    • You can display custom fields as well as predefined fields:
    • id - post ID
    • slug - post slug
    • post-type - post type
    • post-status - post status
    • url - post URL
    • edit-url - post edit URL
    • title - post title
    • title-link - post title with link to the post
    • title-link-out - post title with link to the post, in new tab: target="_blank"
    • excerpt - post excerpt
    • date - published date
    • modified - last modified date
    • image - featured image
    • image-url - featured image URL
    • image-title - featured image title
    • image-caption - featured image caption
    • image-alt - featured image alternate text
    • image-description - featured image description
    • image-link - featured image with link to the post
    • image-link-out - featured image with link to the post, in new tab: target="_blank"
    • thumbnail - featured image thumbnail
    • thumbnail-link - featured image thumbnail with link to the post
    • author - post author
    • author-id - post author ID
    • author-url - post author URL
    • avatar - post author avatar

More parameters

Formatting

  • words or length - limit output by number of words or characters
  • class - add <div> class to the output
  • format - add paragraph tags <p> and preserve line breaks <br> - set to true/false - by default, post content is formatted and for all other fields, it's set to false
  • filter - set true to apply the_content filter; this might be useful when using plugins that filter the post content, for example, Page Builder
  • embed - auto-embed URLs like YouTube, Vimeo, etc - set to true/false - by default, such URLs in post content are auto-embedded
  • currency - format as currency, for example currency="usd"; for supported currencies and custom formatting, see this section at the bottom of this page
  • date_format - use a custom format when displaying field="date", "modified", or a custom field
    • For example, "d.m.Y" will display as 18.11.2013. Please refer to the codex: Formatting Date and Time. Note: instead of backslash, use double slashes to escape characters - "Y/m/d //a//t g:i A" will show as 2013/11/17 at 11:06 PM. To format a unix timestamp value, use: in="timestamp"
Read more
  • more - set true to display content up to the <!--more--> tag, or an excerpt if the tag doesn't exist. It will add "Read More" at the end, with a link to the post; to change the text, use: more="..."
  • dots - set false to disable dots at the end of excerpt
  • link - set false to disable link to the post
Image field
  • image - display an image field, for example: image="product_image"
  • size - size of image: thumbnail, medium, large, full (default) or custom defined size
  • width, height - set both to resize image by pixels; set size parameter with same proportion
  • image_class - add class to the <img> tag
  • alt, title - additional <img> attributes
  • in - type of image field: id (default), url, or object
  • nopin - set nopin="nopin" to prevent Pinterest pinning of image

Taxonomy

  • taxonomy - display category, tag, or a custom taxonomy of the post
    • You can also use [taxonomy tax_name].
  • field - taxonomy field to display: name (default), id, slug, description, or custom taxonomy field
  • term - get taxonomy term by ID or slug, regardless of current post
  • term_name - get taxonomy term by name/label

Other field types

  • meta - display author meta: field="author" meta="user_email" - see: Codex: Author Meta
  • out="label" - when displaying checkboxes or select fields made in ACF, this parameter displays their labels instead of values
Other content types
  • area or sidebar - display a widget area/sidebar by title
  • menu - display a menu list by name/slug, title, or ID. Also see: Bootstrap tabs and navbar.

Field

Use [field] as a shorter way to display a field from the current post.

Display fields

The Article <b>[field title]</b> by [field author] was written on [field date]. Instead of [content field="field_name"] you can write [field field_name].

Additional parameters

[field title words="10"]

Display an image field

[field image="image_field" size="thumbnail"]

Array

You can loop through an array of key-value pairs stored in a field, using the [array] shortcode. [array field_name]   [field key_1]   [field key_2] [/array]

Parameters

  • each - set true to loop through multiple arrays of key-value pairs
  • debug - set true to print the whole array and see how it's structured
  • global - access global variable with given name

Currency format

To manually format a currency value, use these parameters.
  • decimals - number of decimal points; default is 2
  • point - separator for the decimal point; default is "."
  • thousands - separator for thousands; default is ","
For example: [field field_name decimals="0" point="," thousands="."]
To use a predefined currency to format, use the parameter currency. Please note that the currency symbol is not included in the output. [field field_name currency="USD"] The following currencies are defined for your convenience.
