Getting started

Here is an example of how to display your content. Let's imagine a bicycle shop. 1. Create a custom post type called product. Add product categories, and custom fields such as model, price, and description. You can use plugins like Custom Post Type UI and Advanced Custom Fields to create your own post types, taxonomies and fields. Themes and plugins often come with built-in content types. To find their exact names to use, see your site's Content Overview. 2. Add all products in the store as new entries, with featured image and other details. 3. Create a new page to display bicycles. A basic template could be: [loop type="product" category="bicycle"]   [field image]   Model: [field model]   Price: [field price]   Description: [field description] [/loop] 4. Another page or section can display a list of bicycles with a freestyle tag: <ul>   [loop type="product" category="bicycle" tag="freestyle"]     <li>       [field title-link]       [field model] - [field price]     </li>   [/loop] </ul>

Editing

You can use these shortcodes inside the post with the Text editor. There are a couple points to keep in mind. 1. The content of the post editor is automatically formatted, which can create unwanted paragraphs and breaklines with multi-line shortcodes. Wrap the section with the [raw] shortcode to prevent this. 2. The Visual mode is not suited for editing HTML, because the code is invisible and easy to accidentally delete. This can be a challenge if the post content must be editable by a client. Possible solutions are:
  • •  Use HTML Block shortcodes, so the tags are clearly visible
  • •  Disable the Visual mode per page or post type, with the plugin Raw HTML
  • •  Move the code outside the post content with one of the methods below
Shortcode templates can be loaded from a number of places.
  • Custom post type: [content type="template" name="home-page"]
  • Custom field: [field code_block shortcode="true"]
  • Sidebar: Enable shortcodes inside Text widget
  • File: [load dir="views" file="recent-posts.html"]

Notes

HTML attribute

When using a shortcode in an HTML attribute, use single quotes for passing parameters. <div class="[taxonomy category field='slug']"> If a field value may include characters like "quotes" and <brackets>, use the escape parameter. <a href="[field url]" title="[field title escape='true']">
