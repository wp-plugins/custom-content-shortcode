Mobile detect

Enable the mobile detect module under Settings. Use the [is] shortcode to display content based on device type. [is mobile]   User is on a phone or tablet. [else]   User is not on a mobile device. [/is] The parameters available are: mobile, phone, tablet, and computer. Using these in combination, you can selectively load images, page sections, stylesheets, etc. You can even redirect users to another URL.

Examples

Image sizes

[is mobile]   [field image size="medium"] [else]   [field image size="large"] [/is]

Stylesheets

[is computer]   [load css="style.css"] [else]   [load css="style-mobile.css"] [/is]

Redirect visitors on mobile

[is mobile]   [redirect][url site]/mobile/[/redirect] [/is] These last two would be placed in a custom field named css, to load in the head of the page.

Body class

There are CSS classes added to the <body> element, for styling purposes: .is_phone, .isnt_phone, .is_tablet, .is_mobile, .is_computer and .isnt_computer.

Library

Device detection is based on a lightweight PHP class, Mobile Detect version 2.8.12.
